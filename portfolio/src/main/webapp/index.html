projects

relational database management system

uses map and multimap data structures based on b+ trees, binary files for disk storage of database record data, text files for storing names of fields, tables, number of records and other information that needs to be remembered from session to session, and a state machine representing the sql-like grammar for command line interpretation.

2d graphing calculator

uses the shunting yard algorithm and stack and queue data structures built from scratch to parse mathematical functions given by the user. uses sfml to display graphs. the user can zoom in and out, and pan horizontally or vertically. supports trig, floor, ceiling, polynomial, rational, max, and absolute value functions.

tweet index

downloads all the tweets of a particular twitter account and uses the bucket sort algorithm to generate an alphabetized index of each word found.
