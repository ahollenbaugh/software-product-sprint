Projects

  • Relational Database Management System - Uses map and multimap data structures based on B+ trees, binary files for disk storage of database record data, text files for storing names of fields, tables, number of records and other information that needs to be remembered from session to session, and a state machine representing the SQL-like grammar for command line interpretation.
  • 2D Graphing Calculator - Uses the shunting yard algorithm and stack and queue data structures built from scratch to parse mathematical functions given by the user. Uses SFML to display graphs. The user can zoom in and out, and pan horizontally or vertically. Supports trig, floor, ceiling, polynomial, rational, max, and absolute value functions.
  • Tweet Index - Downloads all the tweets of a particular Twitter account and uses the bucket sort algorithm to generate an alphabetized index of each word found.
